# Gopher-Nomics Podcast — Analytics Dashboard

Interactive analytics dashboard for the **Gopher-Nomics Podcast** (Season 1), built with HTML and [Chart.js](https://www.chartjs.org/).

## Charts Included

- **KPI Summary** — Total downloads, unique listeners, top episode, mobile %
- **Daily Trend** — Downloads & listeners over time (Nov 2025 – Feb 2026)
- **Episode Performance** — Downloads per episode (horizontal bar)
- **Platform Split** — Mobile vs. Computer (doughnut)
- **Device Breakdown** — iPhone, Android, Windows, Mac (doughnut)
- **Listening Apps** — Apple Podcasts, Spotify, Chrome, and more (bar)

## How to View

1. Clone this repo
2. Open `gopher-nomics-dashboard.html` in any browser — no server required
3. **Or** open in VS Code and use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension

## Data

Raw CSV data files are in the `data/` folder. The dashboard reads from inline data for simplicity (no fetch calls needed).

## Screenshot

Charts are interactive — hover for tooltips. Right-click any chart → *Save image as…* to export for presentations.
